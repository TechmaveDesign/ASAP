Customer Consent Approval Workflow Demo

How it works:
  • When status is "Waiting For Approval" → All buttons are disabled
  • When status changes to "Approved" → "Loan Agreement" button becomes enabled
  • Click enabled button → Shows confirmation modal
  • On confirm → Button changes to "In Progress" with orange color
  • After API simulation (2 seconds) → Button changes to "Completed" with green color
  • Next button in sequence becomes enabled automatically

Integration Instructions:
To integrate into your PHP file:
  1. Add these lines in the <head> section of your PHP file (before closing </head>): 
    <link rel="stylesheet" href="path/to/approval-workflow.css">
  2. Add this line before the closing </body> tag: 
    <script src="path/to/approval-workflow.js"></script>
  3. The workflow buttons will be automatically inserted after the .StatusBox element
  4. When your assessment form is submitted and status changes to "Approved", the first button will automatically enable
API Integration:

To connect with your actual API, modify the simulateAPICall method in approval-workflow.js:

simulateAPICall(button, step) {
  // Replace this with your actual API call
  fetch('/api/update-approval-step', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ step: step, customer_id: 'VASAP202500001' })
  })
  .then(response => response.json())
  .then(data => {
    if (data.success) {
      this.completeProcess(button, step);
    }
  })
  .catch(error => {
    console.error('Error:', error);
    // Handle error
  });
}
